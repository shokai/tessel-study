# tessel study

- http://start.tessel.io
- https://tessel.io/docs

## Install Dependencies

    % brew install node
    % npm install
    % npm install tessel -g

## Setup

    % tessel update
    % tessel blink
    % tessel wifi -n [WiFi AP NAME] -p [PASSWORD] -s wpa2

## REPL

    % tessel repl

## blink LED

    % tessel run blink_led.js

## camera

    % tessel run camera.js --upload-dir .
